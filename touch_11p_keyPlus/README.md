## 程序功能

使用了扩展口A2,通过继电器来控制点击头动作

## 说明：

这个代码的功能是,

开机后，等待A2口对地闭合，当A2口对地开关闭合后，先是J1点击头点击35毫秒，然后等3秒，J2点击头点击屏35毫秒结束本次A2口信号相应

同时等待下一次A2信号触发
